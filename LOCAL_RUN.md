# Rodrigo PRO — Manual de Execução Local

Siga os passos abaixo para rodar o dashboard no seu computador.

### 1. Pré-requisitos
Certifique-se de que você tem o **Node.js** instalado (versão 18 ou superior). Se não tiver, baixe em: [nodejs.org](https://nodejs.org/).

### 2. Abrindo o Terminal
Abra o terminal (PowerShell ou VS Code Terminal) na pasta raiz do projeto:
`c:\DEV\Cliente Stela - RODRIGO`

### 3. Acessando a pasta da aplicação
Digite o comando abaixo para entrar na pasta onde o dashboard está localizado:
```powershell
cd stela-dashboard-app
```

### 4. Instalando dependências (apenas na primeira vez)
Execute o comando abaixo para instalar todas as bibliotecas necessárias:
```powershell
npm install
```

### 5. Iniciando o Dashboard
Para rodar o site em tempo real, use o comando:
```powershell
npm run dev
```

### 6. Visualizando o resultado
O terminal vai mostrar um endereço parecido com este:
👉 `http://localhost:5173/` (ou algo equivalente com a porta 5000)

Basta copiar e colar este link no seu navegador (Chrome, Edge ou Firefox).

---
**Observação**: Se você fizer alterações no código, o navegador atualizará automaticamente para mostrar as mudanças! 🚀
